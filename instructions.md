# Projeto: Glow-Up Content Express
Assistente de marketing com IA para clínicas de estética de luxo.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Estilo: Tailwind CSS + Shadcn/UI
- Ícones: Lucide-react
- IA: Google Gemini 1.5 Flash
- Cores: Dark Mode, Rose Gold (#E7A1B0), Slate Deep.

## Funcionalidades
1. Formulário para descrever procedimento e escolher tom de voz.
2. Integração com API do Gemini para gerar conteúdo.
3. Exibição de 3 cards: Instagram, Reels e WhatsApp com botão de copiar.

## Design
- Estilo Glassmorphism (cartões semi-transparentes).
- Botão principal com efeito "Shimmer" (brilho).
